VARTA AI Customer Agent

A consumer-facing AI agent built for the VARTA automotive battery brand and embedded directly into their public website. Designed to handle customer inquiries for the European and DACH market in both English and German, with a projected $35,000 in annual savings by reducing reliance on a third-party call center vendor.

Role UX/UI Applications & Data Analytics Intern
Company Clarios
Market Europe / DACH Region
Microsoft Copilot Studio Sitefinity CMS Power Automate Azure Blob Storage JavaScript Python CSS

Overview

VARTA, one of Clarios' major European battery brands, was relying on a third-party vendor to handle customer inquiries for the DACH region via a phone hotline at a cost of approximately $40,000 per year covering product questions, warranty inquiries, and workshop support.

I built a Copilot Studio AI agent embedded directly into the VARTA website to handle these inquiries automatically. The agent supports customers in English and German, can decode battery manufacturing codes, and escalates complex issues to a VARTA specialist via email. At an estimated $400 per month to run, the agent was built with a projected $35,000 in annual savings as the core business case - pending production deployment.

The Problem

High Vendor Cost

The existing third-party hotline cost approximately $40,000 per year to handle inquiries that an AI agent could resolve automatically at a fraction of the cost.

Multilingual Market

The DACH region requires both English and German support, creating complexity for a single agent to serve all customers seamlessly.

External Deployment Complexity

Embedding a Microsoft-hosted AI agent into a third-party CMS (Sitefinity) introduced security and browser policy challenges that had to be solved before anything could go live.

What I Built

01

Custom Embedded Chat Widget

Built a custom JavaScript chat widget from scratch rather than using the default Copilot Studio embed. The widget uses a floating toggle button with VARTA-branded icons hosted in Azure Blob Storage. When a user opens the chat, the VARTA logo icon transitions to a downward arrow indicating it can be clicked again to close - with smooth CSS cubic-bezier animations on open and close for a polished feel. The widget was designed to match VARTA's brand identity while keeping the experience intuitive.

02

CSP Blocking Resolution

When first deploying the widget into Sitefinity, the chat interface failed to load due to Content Security Policy (CSP) restrictions blocking Microsoft-hosted domains. I identified the specific domains causing the block and worked with the web admin to whitelist them in the correct sequence: first cdn.botframework.com for the WebChat library, then *.api.powerplatform.com for the Copilot Studio token endpoint. Solving this unblocked the entire deployment.

03

English / German Auto-Detection

Configured the agent to automatically detect whether a user is writing in English or German and respond in the same language - all handled through system instructions in Copilot Studio without any manual language switching required from the user. This was critical for serving VARTA's DACH customer base naturally.

04

Battery Manufacturing Code Decoder

Built a feature that lets customers type a 13-digit battery code directly into the chat and receive the manufacturing month and year decoded from it. The decoding logic was written in Python and runs inside a Copilot Studio prompt tool - the agent passes the code to the tool, the Python processes it, and the result is returned inline in the conversation. This gives customers instant answers to a question that previously required calling the support hotline.

05

Specialist Escalation Flow

Designed a Power Automate escalation flow that triggers when the agent cannot resolve a customer's inquiry. The flow sends a structured email to the VARTA specialist contact with the customer's issue details for follow-up. Pending a final admin configuration to enable anonymous external users to trigger the Send an Email V2 action without Microsoft OAuth authentication.

06

Dev-to-Prod Deployment Pipeline

Set up a structured deployment pipeline using Power Platform solutions to manage the promotion of agent updates from the development environment to production safely. This ensures changes can be tested and validated before reaching the live customer-facing site.

Technical Details

Python in Copilot Studio Prompt Tools

The battery code decoder uses Python running inside a Copilot Studio prompt tool - an approach that keeps the logic self-contained within the agent without requiring an external Azure Function or API call. The agent passes the 13-digit code as input, the Python extracts the manufacturing date, and the result feeds directly back into the conversation.

Azure Blob Storage for Brand Assets

VARTA-branded icons for both the floating button and the in-chat bot avatar are hosted in Azure Blob Storage. Using Azure Blob kept the assets on Microsoft-controlled infrastructure, avoiding additional CSP domain issues that would have occurred with third-party image hosting.

Generative Orchestration Mode

The agent was designed to use Copilot Studio's generative orchestration mode rather than rigid predefined topic flows. This lets the agent reason about which capability to invoke based on what the user says, making conversations feel natural rather than scripted.
