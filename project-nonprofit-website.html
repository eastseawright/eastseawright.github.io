Crash Dive Base Submarine Veterans Website

Redesigned a submarine veterans organization's website as part of a four-person team. We added security certificates, updated the backend, and reorganized the navigation so veterans and their families could actually find what they needed.

Role Web Development Team Member
Organization UWM Nonprof-IT
WordPress PHP SSL/TLS UX/UI Design Agile Methodology

Project Overview

Crash Dive Base is part of the United States Submarine Veterans Inc., a nonprofit dedicated to honoring submarine service members and engaging the veteran community. Their website needed updating to better serve their members and visitors.

Our four-person team (East, Ayomide, Connor, and Alex) through the UWM Nonprof-IT program worked together to redesign the site — improving security, cleaning up navigation, and making it easier for the veteran community to stay connected.

Challenges

No Security Certificate

The site didn't have SSL, so browsers showed it as "not secure" — not a good look for an organization asking people to trust them.

Confusing Navigation

The menu was disorganized, making it hard for veterans and visitors to find basic information about the organization.

Outdated Backend

The site was running an old version of PHP, which created security risks and blocked us from adding modern features.

Diverse Audience

The site needed to work for active veteran members, family members, and people learning about the organization for the first time.

Approach & Methodology

01

Homepage Redesign

Redesigned the homepage to clearly communicate the organization's mission and put the most important information up front. Improved the visual layout to make a strong first impression for new visitors.

02

Contact Page Improvements

Made it easier for veterans, potential members, and community members to get in touch. Improved the contact form and made key contact information more visible.

03

Navigation Cleanup

Reorganized the menu around what visitors actually need — membership info, events, and veteran resources. Grouped related pages together and reduced the number of clicks to reach important content.

04

PHP Update

Updated the site from an outdated PHP version to a modern one. This was a necessary first step before we could add the security certificate, and it also made the site load faster.

05

SSL Security Certificate

Added an SSL certificate so the site now uses HTTPS — encrypting visitor data, showing the padlock icon in browsers, and preventing attackers from creating fake versions of the site.

Key Features Delivered

Secure Browsing

With the PHP update and SSL certificate, all visitor activity on the site is now encrypted. Browsers show the padlock icon, which builds trust with the community.

Clear Navigation

Reorganized the menu so veterans and visitors can find membership info, events, and community resources with fewer clicks and less confusion.

Updated Design

Refreshed the homepage and contact pages with a modern look that still respects the organization's military heritage and history.

Faster Load Times

The PHP update also improved page speed, making the site work better for everyone, especially visitors with slower internet connections.

Team Collaboration & Learning

  • Security Basics Matter: Learned hands-on how SSL certificates work and why they depend on having updated backend software. Small security steps make a big difference in visitor trust.
  • Working with Non-Technical Clients: The veteran organization had specific needs and limited tech resources. I learned to explain technical concepts in plain language and focus on what mattered to them.
  • Navigation Design Matters More Than You Think: Reorganizing a menu sounds simple, but it dramatically changed how easy the site was to use. Site structure is a huge part of user experience.
  • Team Coordination: Worked with three teammates using Agile methods — dividing work, reviewing each other's contributions, and integrating everything into a cohesive final product.
  • Purpose-Driven Work: Building for a submarine veterans organization reinforced that technology serves real people and communities. This project directly supports veterans' ability to connect and honor their service.
