Crash Dive Base Submarine Veterans Website

Complete website redesign for a nonprofit submarine veterans organization, improving navigation structure, implementing modern security protocols including SSL certificates, and updating the technical infrastructure to better serve the veteran community.

Role Web Development Team Member
Timeline 2022-2023
Organization UWM Nonprof-IT
WordPress PHP SSL/TLS UX/UI Design Agile Methodology

Project Overview

Crash Dive Base, USSVI (United States Submarine Veterans Inc.) is a nonprofit organization dedicated to perpetuating the memory of shipmates and engaging the community as veterans. Their existing website, maintained by webmaster Manny Garmendez, needed modernization to better serve the veteran community.

As part of the UWM Nonprof-IT program, our four-person team (East, Ayomide, Connor, and Alex) worked collaboratively to redesign the website, focusing on improved navigation, enhanced security, and better user experience for the submarine veteran community and their families.

Challenges

Outdated Security

The site lacked SSL certificate implementation, leaving user data vulnerable and affecting trustworthiness.

Navigation Complexity

The primary navigation menu was poorly organized, making it difficult for veterans and visitors to find key information.

Legacy PHP Version

Outdated PHP version created security vulnerabilities and prevented implementation of modern security features.

Veteran Community Needs

The site needed to serve diverse users—from active veteran members to family members seeking information about the organization.

Approach & Methodology

01

Homepage Redesign

Completely redesigned the homepage to better communicate the organization's mission and make key information immediately accessible. Improved visual hierarchy to highlight the veteran community's purpose of perpetuating shipmate memory and community engagement.

02

Contact Page Enhancement

Restructured the contact page to make it easier for veterans, potential members, and community members to connect with Crash Dive Base. Improved form usability and contact information visibility.

03

Navigation Menu Reorganization

Completely restructured the primary navigation menu based on user needs and common tasks. Grouped related content logically, reduced menu depth, and created clearer pathways to important information about membership, events, and veteran resources.

04

PHP Version Update

Updated the site from legacy PHP to a modern version, which was a critical prerequisite for implementing SSL certificates. This update improved site performance, security, and enabled modern security features to protect user data.

05

SSL Certificate Implementation

Implemented SSL/TLS certificate to enable HTTPS, keeping user data secure and verifying website ownership. This prevents attackers from creating fake versions of the site and provides visual trust indicators (padlock icon) to visitors. The SSL certificate ensures encrypted communication between users and the site.

Key Features Delivered

Enhanced Security

Upgraded PHP version and implemented SSL certificate, providing HTTPS encryption for all user interactions. This protects veteran members' data and prevents malicious actors from creating fake versions of the site.

Improved Navigation

Reorganized primary navigation menu with logical grouping and reduced complexity. Veterans and visitors can now find membership information, event details, and community resources with fewer clicks.

Modern Design

Updated homepage and contact pages with contemporary design that maintains respect for the organization's military heritage while improving usability and visual appeal.

Performance Improvements

PHP update resulted in faster page load times and better overall site performance, improving the experience for all users, especially those with slower internet connections.
