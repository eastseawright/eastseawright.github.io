Enterprise Application Development

Application Catalog

A comprehensive global application management system built with Microsoft Power Apps, enabling cross-team collaboration and advanced filtering capabilities for Clarios' international IT operations.

Role UX/UI Applications Intern
Timeline 2024
Company Clarios
Microsoft Power Apps SharePoint Power Automate Enterprise Architecture

Overview

At Clarios, managing hundreds of applications across global teams was becoming increasingly complex. Different departments needed to track application ownership, licensing, security compliance, and usage patterns. The existing systems were fragmented across spreadsheets and disparate databases.

I designed and developed a centralized Application Catalog using Microsoft Power Apps that serves as the single source of truth for all enterprise applications across the organization.

The Challenge

Data Fragmentation

Application data was scattered across multiple systems, making it difficult to get a complete picture of the IT landscape.

Access Control

Different stakeholders needed different levels of access and filtering capabilities based on their roles and regions.

Scalability

The solution needed to handle hundreds of applications and scale with the company's growth across multiple countries.

Solution & Implementation

I built a Power Apps canvas application integrated with SharePoint as the backend data source. The application features:

01

Advanced Filtering System

Implemented multi-level filtering allowing users to search and filter applications by department, region, owner, license type, security classification, and custom metadata fields. The filtering logic handles complex Boolean operations and provides real-time results.

02

Role-Based Access Control

Designed granular permissions system ensuring users only see applications relevant to their region and department while maintaining global visibility for IT leadership.

03

Collaborative Workflows

Integrated Power Automate flows for application approval processes, license renewal notifications, and automated compliance checks.

04

Dashboard & Analytics

Created comprehensive dashboards showing application portfolio health, licensing costs, security compliance status, and usage patterns across the organization.

Technical Highlights

Power Apps Formula Optimization

Developed complex formulas for efficient data filtering and search functionality. Optimized delegation to handle large datasets without performance degradation.

SharePoint Integration

Designed a normalized SharePoint list structure with proper relationships and lookup columns. Implemented custom views and content types for better data organization.

User Experience Design

Created intuitive UI components with consistent design patterns. Implemented responsive layouts that work across desktop, tablet, and mobile devices.

The Application Catalog has become an essential tool for IT governance at Clarios, enabling better decision-making around application investments, security compliance, and license optimization. The centralized system has significantly reduced time spent searching for application information and improved collaboration between global teams.

Key Learnings

  • Enterprise Scale Complexity: Working with a global organization taught me the importance of designing for scale from day one. Edge cases and regional variations need to be considered early in the architecture.
  • Stakeholder Communication: Regular check-ins with IT teams across different time zones required clear documentation and asynchronous communication strategies.
  • Power Platform Best Practices: Learned advanced Power Apps patterns for performance optimization, especially around delegation limits and formula complexity.
  • Change Management: Successfully rolling out new tools requires training, documentation, and ongoing support to ensure adoption.
