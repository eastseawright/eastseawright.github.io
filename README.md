## üìù How to Deploy to GitHub Pages

### Step 1: Upload Files to GitHub
1. Go to your repository: `https://github.com/eastseawright/eastseawright.github.io`
2. Click "Add file" ‚Üí "Upload files"
3. Drag and drop ALL the files from this folder
4. Scroll down and click "Commit changes"

### Step 2: Enable GitHub Pages (if not already done)
1. Go to your repository Settings
2. Click "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click "Save"

### Step 3: Wait for Deployment
- GitHub will automatically build and deploy your site
- This usually takes 1-3 minutes
- You'll see a green checkmark when it's ready
- Visit: `https://eastseawright.github.io`

## üîß Customization Guide

### Adding a Photo
1. Upload your headshot to the repository
2. In `index.html`, find the hero section
3. Add this code after the `hero-label` div:
```html
<img src="your-photo.jpg" alt="East Seawright" class="hero-photo">
```

### Embedding PowerPoint Presentations

#### Option 1: Google Slides (Recommended)
1. Upload your PowerPoint to Google Drive
2. Open in Google Slides
3. Click "File" ‚Üí "Publish to web"
4. Choose "Embed" tab
5. Copy the iframe code
6. In the project page HTML, find the `presentation-placeholder` div
7. Replace it with your iframe code

#### Option 2: SlideShare
1. Upload to SlideShare.net
2. Click "Share" and copy the embed code
3. Paste in place of the placeholder

#### Option 3: OneDrive
1. Upload PowerPoint to OneDrive
2. Right-click ‚Üí "Embed"
3. Copy the iframe code
4. Paste in the presentation section

## üÜò Troubleshooting

**Site not showing up?**
- Wait 3-5 minutes after pushing files
- Check Settings ‚Üí Pages to ensure it's enabled
- Make sure files are in the root directory, not a folder

**CSS not loading?**
- Check that all files are in the same directory
- Verify file names match exactly (case-sensitive)

**Want to make changes?**
- Edit files directly on GitHub (click the file ‚Üí Edit button)
- Or download, edit locally, and re-upload
