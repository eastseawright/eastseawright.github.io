AI & Cloud Architecture

YouTube Comment Sentiment Analysis

End-to-end cloud solution leveraging Azure services to transform unstructured YouTube comments into actionable business intelligence for Optima batteries. Built a complete data pipeline from comment retrieval through AI analysis to Power BI visualization.

Role UX/UI Applications Intern
Timeline 2024
Company Clarios
Azure Functions Azure Cognitive Services Cosmos DB Power BI Python YouTube API

Project Overview

Customer feedback on YouTube provides valuable insights into product performance, user satisfaction, and emerging issues. However, this unstructured feedback was previously untapped—thousands of comments containing rich sentiment data that could inform product development and marketing strategies.

I architected and built a complete cloud-based sentiment analysis pipeline that automatically retrieves YouTube comments from Optima battery videos, analyzes sentiment using Azure AI, stores results in a scalable database, and visualizes insights through Power BI dashboards.

Business Value

Gain Customer Insight

Identify trends in consumer needs, preferences, and pain points directly from authentic user feedback.

Aid Product Development

Use sentiment analysis to inform product improvements, feature prioritization, and quality initiatives.

Monitor Brand Perception

Track how customers feel about Optima batteries over time and identify emerging issues before they escalate.

Architecture & Implementation

01

YouTube Comment Retrieval

Built an Azure Function App that connects to YouTube's API to automatically retrieve comments from Optima battery video playlists. Configured the function with API keys and playlist URLs, implementing logic to specify how to extract comments efficiently. Organized comment data into structured format for downstream processing.

02

Azure Language Service Integration

Connected Azure Cognitive Services Language API to analyze each comment's sentiment. The service classifies comments as positive, neutral, negative, or mixed, providing confidence scores for each classification. Implemented opinion mining to extract specific aspects of feedback (e.g., battery life, durability, price) and their associated sentiments.

03

Cosmos DB Storage

Moved enriched comments from the Function App to Cosmos DB, a globally distributed, multi-model database. Cosmos DB provides scalable, efficient data storage that can handle growing volumes of comments while maintaining fast query performance for analysis and reporting.

04

Power BI Visualization

Created comprehensive Power BI dashboards that connect to Cosmos DB to visualize sentiment trends over time, identify common topics in feedback, track sentiment distribution, and highlight influential comments. Stakeholders can filter by date range, product model, or sentiment type to drill into specific insights.

Technical Architecture

Serverless Architecture

Used Azure Functions for serverless compute, eliminating infrastructure management and enabling automatic scaling. Functions trigger on schedule or API calls to retrieve and process comments continuously.

AI-Powered Analysis

Leveraged Azure Cognitive Services' pre-trained language models for sentiment analysis, avoiding the need to train custom models. Opinion mining extracts granular insights about specific product features mentioned in comments.

Scalable Data Storage

Cosmos DB's NoSQL architecture handles unstructured comment data naturally while providing global distribution and automatic indexing for fast queries as data volume grows.

End-to-End Pipeline

Designed a complete data pipeline using Python to orchestrate retrieval, analysis, and storage. Error handling and logging ensure reliability, while modular design enables easy extension to other data sources.
