AI & Cloud Architecture

YouTube Comment Sentiment Analysis

An automated system that collects YouTube comments from Optima battery videos, analyzes customer sentiment using Azure AI, and displays the results in a Power BI dashboard — turning thousands of unread comments into useful business insights.

Role UX/UI Applications Intern
Company Clarios
Azure Functions Azure Cognitive Services Cosmos DB Power BI Python YouTube API

Project Overview

Optima battery videos on YouTube have thousands of comments — real customers sharing what they like, what they don't, and what they wish was different. But nobody at the company was reading them in any organized way.

I built a cloud-based system that automatically pulls in those comments, uses AI to classify them as positive, negative, or neutral, and shows the trends in a dashboard the marketing and product teams can actually use.

Business Value

Understand Customers

See what real customers are saying about the product — what they love, what frustrates them, and what they want improved.

Improve Products

Use customer feedback trends to guide product improvements and decide what to focus on next.

Track Brand Perception

Monitor how customers feel about Optima batteries over time and catch problems before they grow.

Architecture & Implementation

01

Pull in YouTube Comments

Built an Azure Function that connects to YouTube's API and automatically collects comments from Optima battery video playlists. The function organizes the raw comment data into a clean format for the next step.

02

Analyze Sentiment with AI

Connected Azure's Language AI service to classify each comment as positive, neutral, negative, or mixed, with confidence scores. The AI also picks out specific topics people mention — like battery life, durability, or price — and how they feel about each one.

03

Store Results in Cosmos DB

All the analyzed comments get stored in Cosmos DB, a cloud database that can handle growing volumes of data while keeping queries fast. As more comments come in over time, the system scales automatically.

04

Visualize in Power BI

Created dashboards that show sentiment trends over time, common topics in feedback, and notable comments. Stakeholders can filter by date, product model, or sentiment type to dig into specific questions.

Technical Architecture

Runs on Its Own

Used Azure Functions so there's no server to manage. The system runs on a schedule to pull and process new comments automatically.

Pre-Built AI Models

Used Azure's pre-trained language models instead of building a custom one from scratch. The AI identifies not just overall sentiment but specific product features people mention.

Flexible Database

Cosmos DB handles the varied structure of YouTube comments naturally and automatically indexes everything so queries stay fast as data grows.

Complete Data Flow

Built the entire pipeline in Python — from pulling comments, to running AI analysis, to storing results. Includes error handling and logging so issues are easy to diagnose.
